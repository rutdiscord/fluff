# Privacy Policy for Fluff
 
**Effective date:** June 29, 2026

**Last updated:** June 29, 2026
 
This privacy policy explains what data Fluff collects, how it is used, how it is stored, and how you can have it deleted. By using the bot, you agree to the practices described here. 
 
## 1. Data We Collect
 
The bot only collects data as necessary to provide its moderation and utility features. Depending on which features a server uses, this may include:
 
- **Discord identifiers** - User/snowflake ID's, guild ID's, channel ID's, and role ID's.
- **Timestamps** - The seven most recent timestamps for when a user violated a ping reply preference are stored entirely in memory, do not persist on bot restart, and are removed within a short period of time.
- **Server configuration** - Settings, whitelists, snippet text, and other configurations set by server administrators.

The bot does **not** collect real names, email addresses, passwords, login credentials, or any sort of personally identifiable information.
 
## 2. How We Use Data
 
Collected data is used to:
 
- Operate and provide the Bot's moderation and utility features.
- Maintain the state of ongoing moderation processes (for example, an active rulepush session).
- Enforce server rules as configured by server administrators.
- Diagnose errors and maintain the reliability of the bot.
 
## 3. How Data Is Stored
 
Data is stored in two ways. The first is a series of json files that the bot reads and writes to, and the second is a database. Both the json files and the database are hosted locally on the same server that runs the bot.
 
## 4. Data Sharing
 
We do **not** sell, rent, or otherwise distribute any of the data stored by the bot. We also do not send or distribute any of the bots data in any sort of AI or machine learning systems.

## 5. Data Retention
 
Data is retained only for as long as it is needed to provide the bot's features. For example, moderation session data is removed when the relevant process concludes.
 
## 6. Your Rights and Choices
 
You have the right to request access to, correction of, or deletion of the data the bot holds about you. To make a request, server members can contact the administrators of the server.
 
## 8. Changes to This Policy
 
We may update this privacy policy from time to time. Material changes will be reflected by updating the "Last updated" date above. Your continued use of the bot after changes take effect constitutes acceptance of the revised policy.
 
## 9. Contact
 
For questions about this privacy policy, or to make a data request, please join the undertale discord server and request to speak with staff.
